03

SQL (Structure Query Language) es un lenguaje de programación para trabajar con bases de datos relacionales como MySQL, Oracle, etc.

MySQL es un intérprete de SQL y funciona como un servidor de bases de datos.

MySQL permite crear bases de datos, tablas, insertar datos, modificarlos, eliminarlos, hacer consultas y muchas operaciones más.

Este tutorial tiene como objetivo introducir los conceptos iniciales del mundo de las bases de datos.

04

Tipos de datos en MySQL

varchar

Se usa para almacenar cadenas de caracteres. Una cadena es una secuencia de caracteres entre comillas simples, por ejemplo: 'Hola'.

"varchar" define una cadena de longitud variable con un máximo de 255 caracteres. Por ejemplo, varchar(30) permite almacenar cadenas de hasta 30 caracteres.

int

Se usa para guardar valores numéricos enteros entre -2,000,000,000 y 2,000,000,000 aproximadamente.

Se utiliza para representar cantidades, precios, entre otros valores numéricos enteros.

05

Tipos de datos en MySQL

TEXTO: Para almacenar texto se usan cadenas de caracteres entre comillas simples. Permiten guardar dígitos sin uso matemático, como códigos, documentos, teléfonos. Tipos: varchar, char y text.

NÚMEROS: Se usan para representar enteros, negativos y decimales. Para valores enteros (cantidades, unidades, precios) se usa integer. Para decimales se utilizan float o decimal.
